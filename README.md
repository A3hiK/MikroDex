## рҹ–ҘпёҸ Overview

**MikroDex** brings a professional, glassmorphism-powered interface to MikroTik hotspot management. Whether you're an ISP technician or a network admin, MikroDex gives you full control вҖ” beautifully.

<br/>

### рҹ’Ў Why MikroDex?

| Problem | MikroDex Solution |
|---|---|
| рҹҳӨ Winbox is outdated & clunky | вң… Modern dark UI with smooth animations |
| рҹҳӨ No real-time monitoring | вң… Live dashboard, auto-refresh every 10s |
| рҹҳӨ No local data history | вң… SQLite вҖ” full session & usage history |
| рҹҳӨ Slow user management | вң… One-click create, edit & delete users |
| рҹҳӨ No offline access | вң… Works fully offline, no cloud needed |

## рҹҡҖ Getting Started

### вң… Prerequisites

- **Node.js** v16 or higher вҶ’ [Download](https://nodejs.org/)
- **npm** or **yarn**
- A **MikroTik router** with REST API enabled on port `8728`

<br/>

### рҹ“Ұ Installation

**1. Clone the repository**

```bash
git clone https://github.com/A3hiK/MikroDex.git
cd MikroDex
```

**2. Install dependencies**

```bash
npm install
```

**3. Start in development mode**

```bash
npm run electron-dev
```

> рҹ’Ў This starts the Vite dev server on `http://localhost:5173` and launches the Electron app with hot reload.

**4. Build for production**

```bash
npm run electron-build
```

> рҹ“Ұ Windows installer will be generated in the `out/` directory.

## рҹ”Ң MikroTik API Usage

```js
import MikroTikService from './services/MikroTikService';

// Connect to router
const result = await MikroTikService.connect('192.168.88.1', 'admin', 'password');

// Get all online users
const onlineUsers = await MikroTikService.getOnlineUsers();

// Create a new hotspot user
const newUser = await MikroTikService.createHotspotUser({
  name: 'testuser',
  password: 'pass123',
  profile: 'default'
});

// Delete a user
await MikroTikService.deleteHotspotUser('testuser');
```

**Supported Operations:**

| Operation | Description |
|-----------|------------|
| `connect()` | Establish connection to router |
| `disconnect()` | Close router connection |
| `getHotspotUsers()` | List all hotspot users |
| `createHotspotUser()` | Add a new user |
| `deleteHotspotUser()` | Remove a user |
| `getOnlineUsers()` | Get live active sessions |
| `getRouterInfo()` | Fetch router system details |

## рҹҗӣ Troubleshooting

<details>
<summary><b>рҹ”ҙ Connection Issues</b></summary>
<br/>

- Verify the router IP address is reachable on your network
- Ensure MikroTik REST API is enabled on port `8728`
  - Go to: **Winbox вҶ’ IP вҶ’ Services вҶ’ api-ssl / api**
- Check firewall rules on both the router and your machine
- Confirm your username and password are correct

</details>

<details>
<summary><b>рҹ”ҙ Database Issues</b></summary>
<br/>

- To fully reset the database, delete this file:
  ```
  C:\Users\{Username}\.mikrodex\database.db
  ```
- Rebuild the native module:
  ```bash
  npm rebuild better-sqlite3
  ```

</details>

<details>
<summary><b>рҹ”ҙ Electron / Build Issues</b></summary>
<br/>

- Clear the Vite cache вҖ” delete the `.vite` folder
- Reinstall all dependencies from scratch:
  ```bash
  rm -rf node_modules
  npm install
  ```

</details>
